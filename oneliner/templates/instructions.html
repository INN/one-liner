How To Setup One Liner

Step one: Login or register

Step two: If registering for the first time, fill out a basic survey. We'll use this to help recommend services that are best suited to your needs and your budget.

Step three: Add and configure services that you want to use on your site.

Step four: Go to your home page in the One Liner app and copy the script from the “Your One Liner” box. Add this code to your website on every page you wanted to be tracked by the services you have selected. The One Liner code should typically added just before the closing tag in the header section of your html page.

All done! One Liner is now set up to manage all of the services you've selected.
